## 项目简介

**YoloStudio** 是一款基于 PySide6 和 Ultralytics YOLO 的可视化目标检测训练工具，提供数据准备、模型训练、预测推理的一站式解决方案。

### 核心特性
- 🎨 **双主题支持**：暗色/亮色主题切换
- 📊 **数据可视化**：数据集统计、类别分布图表
- 🚀 **一键训练**：自动生成训练命令，实时日志监控
- 🎯 **实时推理**：支持图片/视频/摄像头/屏幕多种输入源

## 安装与启动

### 环境要求
- Python 3.10+
- Conda 虚拟环境 `yolodo`
- CUDA (可选，用于 GPU 加速)

### 启动步骤
```bash
# 1. 激活虚拟环境
conda activate yolodo

# 2. 运行程序
python main.py
```

## 功能模块

### 📁 数据准备

#### 功能概述
管理和处理 YOLO 格式数据集，包括统计、编辑、划分等功能。

#### 使用步骤
1. **设置路径**：选择图片目录和标签目录
2. **加载类别**：可选加载 `classes.txt` 文件
3. **扫描数据集**：点击"扫描数据集"分析数据

#### 子功能标签页

| 标签 | 功能说明 |
|------|----------|
| 📊 统计 | 显示类别名称、数量、占比统计表 |
| ✏️ 编辑 | 标签替换、删除、筛选等批量操作 |
| 📂 划分 | 按比例划分训练集/验证集 |
| 📄 YAML | 生成 YOLO 数据配置文件 |

#### 划分参数
- **划分比例**：拖动滑块设置训练集占比（50%-95%）
- **随机种子**：控制划分随机性，相同种子产生相同结果
- **忽略无标签图片**：勾选后跳过无标签的图片

### 🚀 模型训练

#### 功能概述
配置训练参数并启动 YOLO 模型训练，实时显示训练日志。

#### 基础参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| Python | Conda 虚拟环境 | - |
| 模型 | 预训练模型文件 (.pt) | - |
| 数据 | 数据配置文件 (.yaml) | - |
| Epochs | 训练轮数 | 100 |
| Batch | 批次大小 | 16 |
| ImgSz | 图像尺寸 | 640 |
| Workers | 数据加载线程数 | 8 |
| Device | 训练设备 | 0 (GPU) |

#### 高级参数
展开"高级参数"区域可配置更多选项：

**数据增强**：Mosaic、Mixup、Degrees、Scale、Translate、Shear、FlipLR、FlipUD、HSV_H/S/V

**优化器**：Optimizer、LR、Final LR、Momentum、Weight Decay

**训练控制**：Patience、CloseMos、Cache、Multi-scale、AMP

> 💡 **提示**：只有勾选的参数才会加入训练命令

#### 使用步骤
1. 点击"扫描"选择 Python 环境
2. 选择模型和数据配置文件
3. 调整参数并展开高级参数（可选）
4. 确认命令预览
5. 点击"▶ 开始训练"

### 🎯 预测推理

#### 功能概述
使用训练好的模型进行目标检测推理。

#### 输入源类型

| 类型 | 说明 |
|------|------|
| 图片 | 选择图片文件进行检测 |
| 视频 | 选择视频文件进行检测 |
| 摄像头 | 使用摄像头实时检测 |
| 屏幕 | 捕获屏幕区域进行检测 |

#### 检测参数
- **置信度阈值**：检测置信度过滤（0-1）
- **IOU 阈值**：非极大值抑制阈值（0-1）
- **类别筛选**：只检测特定类别

#### 输出选项
- 保存结果视频
- 保存关键帧
- 生成报告 (JSON)
- 只保存高置信度帧

## 界面说明

### 主题切换
点击右上角 🌙/☀️ 图标切换暗色/亮色主题。

所有数值输入控件（SpinBox、Slider、ComboBox）需要**点击获得焦点后**才能使用鼠标滚轮调整，避免滚动页面时误触。

## 常见问题

### Q: 程序启动后窗口位置异常？
A: 程序每次启动都会以默认大小 (1280×800) 居中显示。

### Q: 训练时出现 CUDA 内存不足？
A: 减小 Batch 值或 ImgSz 值。

### Q: 下拉框/SpinBox 样式显示异常？
A: 确保 `resources/` 目录下的 SVG 图标文件存在。
